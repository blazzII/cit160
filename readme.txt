Chrome Extenstion: Highlighted Text Word Count using a simple, one-click popup
1. Copy all repository files to a directory of choice.
2. In Chrome, navigate to chrome://extensions
3. Check the Developer Mode checkbox if needed (upper right)
4. Click Load unpacked extensions ...
5. Load Dialog: Navigate to the folder location you created
6. OK
runs in developer mode - small icon with blue "1"
